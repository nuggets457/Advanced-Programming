# Smart Campus Management System

A command-line based Smart Campus Management System developed in C++ using Object-Oriented Programming principles.

## Features
- Smart device management
- Room management
- Device activation/deactivation
- Search devices
- File handling
- Polymorphism and inheritance
- Smart pointers

## Technologies
- C++
- STL
- OOP
- File Handling

## Device Types
- Security Camera
- Air Conditioning
- Projector
- Room Lighting
- Door Lock

## How to Run
1. Open the project in Visual Studio
2. Build the solution
3. Run the program
